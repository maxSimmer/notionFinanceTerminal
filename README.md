# Notion Finance Terminal

Notion based finance dashboard maintained by a terminal client built via Python using the Notion MCP and Notion Connections API. More of a personal project for finances and a test project all together.

A test project designed and built with Claude but directed by me. Wanted to see what Claude could do with the Notion MCP. Claude Code helped with some of the code  along with the Notion connection.

---

## Features

- Basic Notion dashboard featuring basic KPI's.
- Connected to mulitple Notion databases (accounts, transactions, recurring bills, goals, budget)
- Python terminal to update and maintain the databases.
-- Ability to add/update accounts, add/update transactions per account, track goals, income, and spending habits. Done via manual entry based.
